# 🎨 Simple Color Changer

A very beginner-friendly web project that changes the background color when you click a button.  
Built with HTML, CSS, and JavaScript.

## 🚀 What does it do?
- Displays a title and a button.
- Each time you click the button, the background color changes to a different color.
- Colors cycle through: light blue, light green, light coral, light yellow, lavender, light pink.

## 🛠️ Technologies used
- HTML5
- CSS3
- JavaScript (DOM events)

## 📁 How to run
1. Download or clone this repository.
2. Open the `color_changer.html` file in any web browser (Chrome, Edge, Firefox, etc.).
3. Click the button and enjoy!

- How to style elements with CSS.
- How to change page content dynamically using JavaScript (event listener).
